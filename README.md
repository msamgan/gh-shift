# gh shift

A simple command line tool to switch between gh (github cli) accounts.

## Installation

```bash
git clone https://github.com/msamgan/gh-shift.git
cd gh-shift
php ghs app:build ghs
```

## Usage

```bash
./builds/ghs --help
```

## Making it global

```bash
sudo mv ./builds/ghs /usr/local/bin/ghs
```
